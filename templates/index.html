Featured Project

AABO – AI Digital Assistant 🤖

AABO is an AI-powered digital assistant kiosk that interacts with users using computer vision and voice communication. It detects human presence, greets users, listens to their queries, and provides intelligent responses in real time.

Features:
• Real-time face detection using OpenCV
• Voice interaction (speech-to-text & text-to-speech)
• Intelligent response system using keyword logic
• Mobile camera integration for live video processing

Tech Stack:
Python, OpenCV, SpeechRecognition, pyttsx3

Use Cases:
Smart campus assistant, digital reception system, and interactive information kiosk.
